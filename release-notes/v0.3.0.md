# v0.3.0 — 2026-03-13

## Highlights
- **Client SDK (consumer path):** new `ai-capabilities/client` helpers for discovery + execution with `fetch` overrides, structured errors, and type-safe responses.
- **Express middleware:** `createAiCapabilitiesMiddleware` exposes `/.well-known`, `/capabilities`, and `/execute` directly inside existing Express/Node apps, including basePath support.
- **Zero-config quick scan:** running `npx ai-capabilities` now chains doctor → inspect → extract → detect-llm → auto-bind (dry run) to summarize capability readiness without modifying code.

## Details
- Added public client exports (`discoverCapabilities`, `executeCapability`, `getWellKnownManifest`) plus fetch injection + error typing.
- Implemented middleware + tests, plus docs covering Express integration, external agents, and standardization updates.
- Added CLI quick scan orchestrator, consumer-facing docs/examples, and express sample app demonstrating safe discovery/execution flows.
- Improved server request parsing to reuse Express-parsed bodies and tightened runtime error mapping.
